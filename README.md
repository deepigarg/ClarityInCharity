# ClarityInCharity
A charity application based on Blockchain.

## Requirements

1. Ganache
2. npm
3. Truffle suite
4. Metamask

## To run the application

1. Clone the repository and cd into the folder.
2. run `npm install`
3. Open Ganache and start a test blockchain
4. run `truffle compile`
5. run `truffle migrate`
6. run `npm run dev`
